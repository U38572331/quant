OpenBB Local Deployment
=======================

Location: %~dp0

To run the OpenBB Terminal (CLI):
Double-click: launch_openbb.bat

To run the OpenBB API Server (Platform):
Double-click: launch_api.bat

Source Code:
The source code is located in the "OpenBB-main" directory.
To develop, activate the environment:
C:\Users\user\AppData\Local\pypoetry\Cache\virtualenvs\openbb-GnnGhCRn-py3.13\Scripts\activate

Documentation:
https://docs.openbb.co
